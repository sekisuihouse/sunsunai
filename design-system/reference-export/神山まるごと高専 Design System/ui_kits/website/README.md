# Website UI Kit — 神山まるごと高専 本体サイト

Recreation of the core (monochrome) site: left-rail shell, home, admission (2027年度入試情報), inquiry.

- `index.html` — interactive click-through (bottom-right switcher + in-page links)
- `HomeScreen.jsx` — hero copy, 学校便り news, photo nav (hover mask), CTA cards
- `AdmissionScreen.jsx` — giant-arc vertical heading, breadcrumb, CTA buttons
- `InquiryScreen.jsx` — HubSpot form placeholder (third-party; left deliberately unstyled)

Known gaps (source was a static mirror): home hero is a faithful reconstruction from copy + motifs, not a pixel copy (the mirrored top page contained 404 content); SNS icons in the footer are typographic placeholders.
