# Campaign LP Kit

Campaign landing pages are a separate theme from the mono core site. Two scopes:

- `campaign-tour` (学校説明会 LP): 水色 #58C3F5 / 黄緑 #92DF55, 40px radius
- `campaign-event` (未来の学校FES): 青 #243FAB / シアン #00A0E9 / ピンク #E4007F / 黄 #FFE100, 50px asymmetric radius

`index.html` toggles between both (bottom-right switcher). Structure is a faithful scaffold of the observed LP anatomy (hero + white rounded sections + pill CTA), not a pixel copy — the LP-specific artwork was not in the provided uploads.
